# Skill Stacking Guide (Frontend)

Simple frontend that suggests complementary skills based on your current skills and career goal. Built with plain HTML, CSS and JavaScript.

Usage
- Open `index.html` in a browser (double-click or use a static server).
- Enter your current skills (comma separated), choose a career goal and proficiency, then click `Suggest Skills`.
- Use `Copy JSON` or `Export CSV` to grab results.

Files
- `index.html` — main UI
- `styles.css` — styling
- `app.js` — suggestion logic and interactivity

Notes
- Suggestions are heuristic and for guidance; improve the mapping and resources as needed.
